# new-bongle

A starter project for [bongle](https://github.com/isaac-mason/bongle), the multiplayer voxel game engine for the web.

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/isaac-mason/new-bongle)

Click the badge above to open this project in a one-click cloud dev environment. It boots a Linux container, installs dependencies, and starts the editor. Open the forwarded `:3002` port in Chrome or Edge (WebGPU required) to start building.

## Run locally

```sh
npm install
npm run edit
```

The editor runs on http://localhost:3002. Edit the game code in `src/` and see your changes live.

## Commands

```sh
npm run edit    # start the editor
npm run build   # build into dist/bundle.zip
npm run start   # serve a built dist/ locally
```
